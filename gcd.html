Greatest Common Divisor

Definition

We have two non-zero integers $a$ and $b$. An integer $d>0$ is said to be the greatest common divisor for $a$ and $b$ (also denoted as $(a, b)=\gcd(a, b)$) if:

  1. $d \mid a$ and $d \mid b$
  2. If $c \mid a$ and $c \mid b$ then $c \leq d$

We can extend the definition to include support for $0$ by observing that every non-zero integer divides zero. So if one of the inputs is $0$, the other value is returned. In case both the inputs are $0$, $0$ is returned.
