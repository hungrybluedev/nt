About

Introduction

This website (also an installable, offline-capable Progressive Web App) contains implementations of commonly-used number theory algorithms.

It was created with the following intentions:

  1. Easily Accessible - from any device, even offline after an initial download.
  2. Easy To Use - making use of good design principles and implementing efficient algorithms.
  3. Dependable - ensuring that the results are correct and the functionality is thoroughly tested.

Instructions For Use

The home page contains a list of all the algorithm implementations. The similar algorithms are grouped into relevant sections (such as Divisibility, Primality, etc.)

Click on the boxed links to navigate to the webpage for that respective algorithm implementation. A brief description of the algorithm will be provided.

Fill in the input boxes appropriately and click on the main red button. If the inputs are valid, then the results will appear below the button.
